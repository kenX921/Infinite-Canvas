# 我的无限画布 (Infinite-Canvas)

**基于 [hero8152/Infinite-Canvas](https://github.com/hero8152/Infinite-Canvas) 二次开发**

支持 ComfyUI、本地模型、各种 AI API 调用

### 主要功能
- 无限画布 + 强大分组管理
- 支持几乎所有 API（修改 .env 即可）
- 本地 ComfyUI 支持
- ModelScope 免费调用

### 快速启动
1. Clone 或下载本仓库
2. 双击 `安装依赖.bat` 安装依赖
3. 在根目录创建 `API/.env` 文件，填入你的 API Keys
4. 双击 `run.bat` 启动
